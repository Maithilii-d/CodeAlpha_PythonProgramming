# Basic Chatbot

## Description
This is a simple rule-based chatbot developed in Python as part of the CodeAlpha Python Programming Internship. The chatbot responds to predefined user inputs using conditional statements.

## Features
- Greets the user
- Responds to common questions
- Ends the conversation when the user types "bye"
- Handles unknown inputs with a default response

## Technologies Used
- Python

## Concepts Used
- Functions
- if-elif statements
- Loops
- Input/Output

## Sample Output

```
==================================================
                BASIC CHATBOT
==================================================
Type 'bye' to end the chat.

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: what is your name
Bot: I am a Basic Chatbot.

You: who made you
Bot: I was created using Python.

You: thank you
Bot: You're welcome!

You: bye
Bot: Goodbye!
```
